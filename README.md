# farmrs-club

## Quick Setup

```bash
git clone https://github.com/nirvanaitsolutions/Farmrs-Club.git
cd Farmrs-Club

yarn
yarn dev
```

## Production build

```bash
git clone https://github.com/nirvanaitsolutions/Farmrs-Club.git
cd Farmrs-Club

yarn
yarn build
yarn start
```
